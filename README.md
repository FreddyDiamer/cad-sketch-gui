# CAD Sketch GUI

GUI-приложение на **Python 3 + PyQt6** для автоматизированного формирования эскизов деталей по изображениям.

Экспорт в DXF создаёт файл `.dxf`.

## Запуск

1) Установите зависимости:

```bash
python3 -m pip install -r requirements.txt
```

2) Запустите приложение:

```bash
python3 main.py
```

## Основные сценарии

- Создание проекта (имя + папка) и сохранение в `.json`.
- Импорт изображения (без SVG/PNG) и отображение в рабочей области.
- Калибровка масштаба (ввод реального расстояния).
- Параметры Canny (пороги) и выделение контуров.
- Построение эскиза и его визуализация.
- Экспорт DXF: создание файла `.dxf`.
- Работа с базой данных: подключение/поиск/удаление.
